Using DoesNotExist

Attempts to get object from database. Model DoesNotExist exception is thrown in case query has no results. The base class of exception is caught and either exist or not exist code is executed. documentation

from django.core.exceptions import ObjectDoesNotExist
try:
  {{model}}.objects.get({{query}})
  {{exist_code}}
except ObjectDoesNotExist:
  {{else_code}}
