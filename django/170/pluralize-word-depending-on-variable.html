Django - Pluralize word depending on variable

Create singular or plural of word depending on size of list. For example "1 product", "2 products"

Using pluralize default suffix

Adds default suffix after word base. Default suffix is 's'.

documentation

{{word_base}}\{\{list|pluralize\}\}

Using pluralize custom suffix

Add custom plural suffix if list is larger than 1.

documentation

{{word_base}}\{\{list|pluralize:"{{plural_suffix}}"\}\}.

Using singular and pluralize suffix

Use singular or plural suffix depending on list size.

documentation

{{word_base}}\{\{list|pluralize:"{{singular_suffix}},{{plural_suffix}}"\}\}
c

Notes

  • Can be used for special cases such as "1 cherry", "2 cherries".
