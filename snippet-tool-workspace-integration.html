Snippet tool workspace integration

For the best coding experience consider using a plugin insert code snippets directly from our site into your own code in your editor.

At the moment sublime text and bash plugins are available. If you have created a plugin for your favorite editor, please let us know and we'll include it in the available plugins list.

If you are eager to use snippets from the site directly in your workspace consider rolling your own plugin.

Available plugins

Sublime text

The sublime text plugin is available on github. With it, you can search and insert snippets from the website directly in sublime with just a few key strides.

Bash

The bash plugin is available on github. Use it to insert common or difficult commands quickly.

Create your own plugin

Chances are a plugin for your favorite editor is not yet available. Please consider creating one yourself. Have a look at the API to see what data is available for you to use. And take a peek at the Sublime text plugin for inspiration.

General flow

No matter the language or editor, all plugins will likely follow the same execution path. The general way to search for a snippet and insert it is as follows:

  1. Prompt user for search query
  2. Request resource '/concept/search/?q="[query]"'
  3. * User picks concept
  4. Request resource '/concept/[id]' to find methods available for concept.
  5. Request multiple resources in single query '/method/set/[id1];[id2];[id3]/'
  6. * User picks method
  7. Request resource '/method/[id]'.
  8. * User picks snippet.
  9. Insert snippet at cursor. Optionally allow changing the parameters to fit the users specific needs.
